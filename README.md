This repository contains code for Group 8 Applied Bioinformatics and Cloud Computing coursework.

Our project is focussed on replicating the results of the Yuping Lin, et al., 2014 paper: 'Leveraging transcription factors to speed cellobiose fermentation by Saccharomyces cerevisiae'.

Lin Y, Chomvong K, Acosta-Sampson L, Estrela R, Galazka JM, Kim SR, et al. Leveraging transcription factors to speed cellobiose fermentation by Saccharomyces cerevisiae. Biotechnology for Biofuels. 2014 Aug 27;7(1).
